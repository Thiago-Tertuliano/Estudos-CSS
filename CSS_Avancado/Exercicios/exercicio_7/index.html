Variáveis CSS em Ação

Clique no botão no cabeçalho para alternar entre os temas claro e escuro. Observe como todas as cores, espaçamentos e tipografia mudam automaticamente usando variáveis CSS.

Exemplo de Código:

:root {
  --bg-primary: #ffffff;
  --text-primary: #333333;
}

body.dark-mode {
  --bg-primary: #1a1a1a;
  --text-primary: #ffffff;
}
