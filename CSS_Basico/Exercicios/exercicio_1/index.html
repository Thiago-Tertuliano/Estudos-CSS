Sobre CSS

CSS (Cascading Style Sheets) é a linguagem que usamos para estilizar páginas web. Ela controla cores, fontes, layout e muito mais!

Neste exercício, você vai praticar os conceitos básicos de CSS: seletores, propriedades, classes e IDs. Experimente modificar os estilos para ver as mudanças!

Tipos de Seletores

Seletor por Tag

Seleciona todos os elementos com a mesma tag HTML.

h1 { color: blue; }

Seletor por Classe

Seleciona elementos com uma classe específica.

.destaque { background: yellow; }

Seletor por ID

Seleciona um elemento único com ID específico.

#especial { font-size: 20px; }
